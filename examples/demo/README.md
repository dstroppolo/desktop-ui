# Desktop UI Demo

Demo application for the Desktop UI library.

## Setup

This demo imports the desktop-ui library from `packages/desktop-ui` (via Vite alias for dev, built package for production).

## Development

From the monorepo root:

```bash
npm install
npm run dev:demo
```

Or from this directory:

```bash
npm run dev
```

The demo will be available at `http://localhost:5173`

## Features Demonstrated

- Multiple draggable windows
- Window resizing
- Window minimizing/restoring
- Taskbar integration
- Theme switching (Windows XP / Windows 98)
- Responsive content inside windows
